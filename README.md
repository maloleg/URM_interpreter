# URM_interpreter
simple URM interpreter

format is:
[COMMAND] [NUMBER] .. [NUMBER]
example:<br>

S 3<br>
J 3 1 5<br>
J 3 2 11<br>
J 1 1 1<br>
J 3 2 8<br>
Z 1<br>
J 1 1 100<br>
Z 1<br>
S 1<br>
J 1 1 100<br>
J 3 1 15<br>
Z 1<br>
S 1<br>
J 1 1 100<br>
Z 1<br>
J 1 1 100<br>
e<br>

("e" for end of input)
